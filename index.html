Grunt-contrib-milkman

Плагин осуществляет запись файлов на удаленный адрес после их обработки. Т.е. после того как все прочие плагины генерируют файлы, данный плагин копирует их в другое место.

Установка

Этот плагин требует node.js и Grunt 
npm install grunt-contrib-milkman

Альфа-версия

Данный плагин используется в единичном проекте, поэтому его функционал не универсален. Но, помощь в доработке не отклоняется.

Настройка

Для корректной работы этот плагин должен быть размещен в конце очереди на выполнение. Пример файла gruntfile.js: 
grunt.initConfig({
	snipper: {
	  development: {
	    files: {
	      '../scripts/': ['./scripts/core.js']
	    }
	  }
	},
	milkman: {
		main: {
			files: {
				'../scripts/core.js': 'my/far/folder/core.js'
			}
		}
	}
});
grunt.loadNpmTasks('grunt-contrib-snipper');
grunt.loadNpmTasks('grunt-contrib-milkman');
grunt.registerTask('default', ['snipper','snipper']);
